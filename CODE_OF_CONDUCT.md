# 社区行为准则

言序社区致力于提供友善、开放、无骚扰的协作环境，不因经验、身份、背景或观点而区别对待参与者。

## 我们鼓励

- 以事实和技术理由讨论问题；
- 尊重不同经验水平，给出可行动的反馈；
- 主动承认错误并帮助修复；
- 将语言、社区和使用者的长期利益置于个人胜负之上。

## 不可接受的行为

- 侮辱、歧视、威胁、骚扰或人身攻击；
- 未经许可公开他人隐私信息；
- 持续打断讨论、恶意灌水或破坏项目协作；
- 在项目空间内进行其他明显不专业的行为。

维护者可以删除不当内容、限制参与或封禁严重违规者。报告请通过仓库的私密安全联络渠道提交；维护者会尽力保护报告者隐私并公平处理。

本准则适用于仓库、Issue、Pull Request、Discussion 及代表项目参与的其他公开空间。
