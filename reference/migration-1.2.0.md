# 言序 1.2.0 迁移

言序 1.2.0 向后兼容 1.1.x。语言规范、YXB、清单、锁文件和原生 ABI 的格式版本均未变化；现有项目无需修改即可继续构建。

## 动态打开网页

桌面应用不再需要通过 UI 包或通用进程权限打开运行时确定的网页。源码引入新的标准模块：

```yanxu
引「标准:桌面」为 桌面；

桌面.打开网页（「https://example.com」）；
```

项目清单须继续显式声明已有的专用权限：

```toml
[权限]
打开外部地址 = true
```

`打开网页`只接受含主机的 HTTP(S) URL。`file:`、`mailto:`、无主机或格式错误的地址会以`DESKTOP_URL`失败；未授权调用仍使用既有权限诊断。WASI 返回`DESKTOP_UNSUPPORTED`。

使用此 API 的包应把最低言序版本设为`>=1.2.0`。该能力不要求`进程 = true`，也不应通过 UI 宿主重复实现。
