# 迁移到言序 1.1.16

1.1.16 为现有`网络`权限列表增加 IPv4 与 IPv6 CIDR 授权。

- 使用普通公网主机的应用无需修改清单。
- 需要访问 VPN fake-IP 或其他敏感网段时，应显式列出最窄的 CIDR，例如：

  ```toml
  [权限]
  网络 = ["*", "198.18.0.0/15"]
  ```

- `*`继续允许普通公网地址，但不会隐式授权敏感地址。
- `本地网络 = true`继续只覆盖回环、RFC1918、IPv4 链路本地、CGNAT、IPv6 ULA 与 IPv6 链路本地；CIDR 不会扩大这一通用能力。
- 嵌入宿主与应用的授权仍取交集。CIDR 与较窄 CIDR、单 IP 或带端口 IP 相交时，只保留较窄范围。
