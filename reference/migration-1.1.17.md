# 迁移到言序 1.1.17

1.1.17 是保持 1.1.x 兼容线的补丁版本。需要访问 VPN fake-IP 或其他特殊网段的应用可以在清单中声明最窄的 CIDR：

```toml
[权限]
网络 = ["*", "198.18.0.0/15"]
```

`*`仍只允许普通公网地址，不会自动授权敏感地址；宿主与应用权限始终取交集。既有单 IP、带端口和`本地网络 = true`语义保持不变。

原生 GUI 宿主在回调中再次请求事件泵时，运行时会将嵌套请求视为已由外层泵覆盖的成功 no-op。应用源码和现有制品无需修改；需要 1.1.17 CIDR 能力的应用应重新构建 Bundle。
