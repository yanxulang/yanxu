# 迁移到言序 1.1.17

1.1.17 是保持 1.1.x 兼容线的补丁版本。需要访问 VPN fake-IP 或其他特殊网段的应用可以在清单中声明最窄的 CIDR：

```toml
[权限]
网络 = ["*", "198.18.0.0/15"]
```

`*`仍只允许普通公网地址，不会自动授权敏感地址；宿主与应用权限始终取交集。既有单 IP、带端口和`本地网络 = true`语义保持不变。

原生 GUI 宿主在回调中再次请求事件泵时，运行时会将嵌套请求视为已由外层泵覆盖的成功 no-op。应用源码和现有制品无需修改；需要 1.1.17 CIDR 能力的应用应重新构建 Bundle。

## 子进程超时

`进程.执行`的超时参数现在接受 1 至 86,400,000 毫秒。旧运行时会拒绝超过 300,000 毫秒的调用；升级后可按检查、测试、构建或打包的预期时长设置更长值。既有不超过 300,000 毫秒的调用无需修改。

嵌入宿主和上层工具应为不同操作选择不超过宿主硬上限的默认值。超时会终止并回收子进程，以`PROCESS_TIMEOUT`失败；标准输出和标准错误各自最多保留 16 MiB，超限以`PROCESS_LIMIT`失败。

## 包与制品

本版本不要求重新生成锁文件或当前格式制品。需要使用新的长时超时范围时，运行环境必须升级到 1.1.17 或更高版本。
