# yanxu-package 1.1.8

## 新功能与修复

- 新建 GUI 清单把最低运行时固定为言序 1.1.8，避免 Windows 应用回落到主线程栈不足的
  1.1.7 运行时。
- 包解析、锁图、权限、归档与原生制品接口均不变；普通非 GUI 清单继续兼容 1.1.7。

## 兼容性与升级

清单格式 2、锁文件格式 2 和公开 Rust API 保持兼容。直接把 Cargo 依赖更新到 1.1.8 并
重新生成 Cargo 锁文件即可；已有言序项目的`言序.lock`无需因共享库版本单独迁移。

## 平台与制品

该库是平台无关的 Rust 源码包，在 Windows、macOS 与 Linux 质量矩阵验收。Release 提供
`yanxu-package-1.1.8.crate`及其 SHA-256 文件；它不包含`yanxu`运行时或桌面原生库。

## 第三方依赖与已知限制

本版本不新增或升级第三方依赖。共享包本身不会改变 Windows PE 栈保留量；GUI 应用仍须
安装言序 1.1.8，并用该运行时重新构建 Windows Bundle。
