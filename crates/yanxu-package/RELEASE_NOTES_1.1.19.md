# yanxu-package 1.1.19

本版本与言序 1.1.19 同步发布，保持清单格式 2、锁文件格式 2 与公开 Rust API 的补丁兼容边界。`yanxu-package` 1.1.18 未形成正式共享包 Release，使用 1.1.17 的项目可直接升级。

- 路径、Git 与注册表依赖继续通过不可变 generation 和已打开目录能力绑定校验结果，缓存损坏时保留旧现场并增量修复。
- 清单、锁文件、辖制清单、注册表索引、模块源码和原生制品使用有界普通文件快照，拒绝链接、重解析点、特殊文件和身份漂移。
- Git 子进程继续受硬截止时间、输出上限、磁盘预算与完整进程树回收约束；Windows 精确输出边界回归不再依赖高开销脚本宿主冷启动。
- 共享包随正式制品提供许可证正文、确定性 CycloneDX 1.5 SBOM、构建元数据、SHA-256 与构建来源证明。
- Rust 最低支持版本为 1.89；既有格式 1/2 清单和锁文件无需迁移。
