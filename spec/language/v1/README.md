# 言序语言规范 1

状态：**1.0 正式规范**。本目录规定言序 1.x 的源代码语义和可观察行为；实现若与本文冲突，以规范及兼容语料为准。勘误可以澄清文字，但 1.x 不得借勘误引入破坏性语义。

规范分为：

- [词法与文法](lexical-and-grammar.md)
- [类型、对象与协议](types-and-objects.md)
- [模块、任务与错误](modules-tasks-errors.md)
- [执行器与兼容性](execution-and-compatibility.md)
- [公共格式](formats.md)

关键词“必须”“不得”表示实现要求；“应”表示除非有明确理由，否则应遵守；“可以”表示可选行为。未规定行为不得被依赖为 1.x 兼容承诺。
