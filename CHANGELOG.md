# 变更记录

## 1.1.4

### 新增

- 新增不可变`字节串`类型与`标准:字节`，覆盖 UTF-8 编解码、实际字节长度、半开切片、拼接、查找和 0..255 数列互转；单值上限为 4 MiB。
- `文件`新增二进制读取、写入、追加与含种类/字节数/只读/修改时间的状态查询。
- `套接字`新增 TCP/UDP 字节收发、TCP 精确读取、显式 EOF、关闭写端与 TCP 无延迟；提前 EOF 使用`SOCKET_EOF`。
- `网络.请求字节`支持自定义请求首部、二进制请求正文和二进制响应，并拒绝 CRLF、非法首部及调用方控制的逐跳/长度首部。
- 新增操作系统安全随机、HMAC-SHA256、常量时间比较、HTTP 日期格式/解析和`环境.参数`；CLI 使用`--`分隔程序参数。

### 工具链

- 树解释器、独立 VM、静态类型、标准库 API 清单、LSP 标准库成员补全、格式化器、文档生成器、Rust/C/WASI 嵌入与兼容语料同步`字节串`契约。
- 嵌入`Execution`与 C ABI 可返回原始字节数列；新增二进制文件、HTTP、TCP EOF/短读、密码向量和双后端一致性回归。
- 本版明确不包含 HTML、HTTP 服务端、Web 框架、真正异步 I/O、服务端 TLS、HTTP/2/3、WebSocket、原生编译器或自举。

## 1.1.3

### 安全

- HTTP 重定向改为逐跳重新校验目标主机，DNS 解析后再校验实际 IP；回环、私网、链路本地与特殊地址须精确授权，防止重定向越权与 DNS rebinding/SSRF。
- 拆分`network`外连、`tcp_listen`监听和`udp_bind`绑定权限；沙箱默认不允许任何监听，包清单新增`TCP监听/UDP绑定`声明。
- 远程包制品改为进程内安全解压：拒绝符号链接、硬链接、设备文件、管道、越界/过长路径和重复写入，并限制压缩体积、单文件、展开总量与条目数。

### 修复

- HTTP、TCP 连接与 UDP 发送的 DNS、多地址尝试和传输共享同一调用级截止时间，不再在 DNS 结束后重置超时。
- TCP 接收保留未完成 UTF-8 字符的内部字节，正确组合跨多次底层读取的中文；每个运行时最多同时持有 128 个套接字资源和 16 个 TCP 监听器。
- 持久`Engine`在运行失败后强制重建运行时并清空类型历史，避免部分状态与静态历史分叉；新增显式`Engine::reset`。
- Release 标题使用已校验版本值，制品全部上传前保持草稿；所有 GitHub Actions 固定到完整提交 SHA。

## 1.1.2

### 新增

- 新增原生`标准:套接字`模块和不可伪造的`套接字`资源值，覆盖 TCP 连接/监听/接受/收发与 UDP 绑定/收发。
- 所有连接和阻塞 I/O 使用调用级超时；单次接收由调用预算和 4 MiB 硬上限共同约束，文字载荷严格校验 UTF-8。
- 新增`本地地址`、`对端地址`和幂等`关闭`；资源随最后一个引用释放时由宿主自动关闭。
- 套接字复用`network`主机白名单并新增稳定`SOCKET_*`错误；捕获后的类别为`套接字`，WASI 返回`SOCKET_UNSUPPORTED`。

### 工具链

- 树解释器、独立 VM、静态类型摘要、机器 API 清单、示例和兼容语料同步套接字公共契约；标准模块增至 22 个。

## 1.1.1

### 新增

- 对象系统新增`父.方法（…）`与`父.初始化（…）`，父类实现使用当前动态实例；树解释器、独立 VM、静态检查器和格式化器保持一致。
- 子类值可用于父类标注；静态检查器验证覆写的成员种类、静/实例属性、可见性及完整法签名。
- 新增`值 是 类型`原生判断，支持联合、可空、容器泛型、法、类与协议，并在`若`分支内收窄简单变量。
- `网络`新增五参数`请求`，返回`状态`、最终`地址`、`首部`与`正文`；原生目标支持 HTTP/HTTPS、分块传输、10 秒默认全程超时与 4 MiB 默认响应上限。
- 网络失败使用稳定`NET_*`代码；捕获后的`误`新增`代码`与`类别`，并继续保留`消息`、`位置`和`踪迹`。

### 工具链

- 同步语言规范 1、Tree-sitter 文法/高亮、兼容语料、对象与类型判断示例及标准库机器清单。
- 新增 HTTPS 双后端实测，以及分块解码、超时、响应上限、权限和网络错误分类回归；WASI 继续保持无隐式网络实现。

## 1.1.0

### 新增

- 新增`Base64`、`正则`、`URL`与`日期`四个确定性、全平台、零宿主权限标准模块，稳定模块总数由 17 增至 21。
- `Base64`提供标准及 URL 安全无填充的 UTF-8 编解码；非法字母表、填充与非 UTF-8 结果均有稳定诊断。
- `正则`提供 Unicode 匹配、首项、全量替换与分割，使用线性时间引擎并明确报告非法模式。
- `URL`提供校验、协议/主机/端口/路径/查询读取与相对地址合并；纯解析不申请网络权限。
- `日期`提供 ISO 公历校验、闰年、加天和日期差，稳定范围为`0001-01-01`至`9999-12-31`。

### 工具链

- 新增`compat/1.1/扩展标准库.yx`与`examples/1.1标准库.yx`，同步共享内核、静态类型摘要、机器 API 清单及树解释器/VM 正向和错误一致性测试。
- CI 将兼容语料门槛提升至 10 卷、标准模块清单提升至 21 项；Tree-sitter 元数据同步到 1.1.0。
- 路径依赖内容哈希稳定排除`.yanxu`和`.DS_Store`工作区制品，避免本机安装目录或系统元数据使锁文件漂移。
