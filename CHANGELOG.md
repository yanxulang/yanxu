# 变更记录

## 1.1.0（开发中）

### 新增

- 提供独立官方包管理器[言包](https://github.com/YanXuLang/yanbao)所需的稳定清单规范化与包名/入口校验 API；运行时继续负责唯一的依赖解析和锁文件实现。
- 在 1.0 稳定基线后新增`随机`、`标识`、`模板`与`校验`四个无宿主权限标准模块，标准模块总数增至 17。
- `随机`使用显式安全整数种子，跨树解释器、VM 与平台生成可复现的小数、半开区间整数和布尔值。
- `标识`提供 SHA-256 派生的版本 8 稳定 UUID、UUID 校验与可配置稳定短码。
- `模板`提供`{{名称}}`单项插值和 HTML 转义/反转义；`校验`覆盖常用电子邮件、IPv4、十六进制色和 Unicode 标识符形式。
- 新增`examples/后1.0标准库.yx`，并同步静态类型摘要、机器 API 清单、双运行时一致性测试、文档站与 VS Code 模块补全。

## 1.0.0

### 新增

- 冻结语言规范 1、包/锁/字节码/JSON 格式 1、1.x SemVer 兼容政策与 YXP 提案流程。
- 新增`异 法`、`任务<T>`、`候`、取消、状态与失败传播的结构化`并候`，并保持树解释器/VM 一致。
- 新增默认沙箱的持久 Rust 嵌入引擎、版本化 JSON C ABI、WASI 入口以及文件/网络/环境/进程权限。
- 新增执行步数、调用深度和集合增长预算，资源耗尽诊断直接指出可调整的配置项。
- 新增 0.3–0.9 兼容语料与`yanxu 兼容`、版本化字节码归档、三类模糊测试入口与固定种子。
- 新增弃用登记和`yanxu 迁`检查/差异/就地迁移，以及`yanxu 标准库 --json`机器 API 审计。
- 新增`路径`、`环境`、`哈希`、`编码`、`统计`与`CSV`六个标准模块，并提供结构化静态类型摘要。
- 新增共享标准库内核与`examples/扩展标准库.yx`，覆盖路径处理、SHA-256、UTF-8 编码、总体统计和 RFC 4180 风格 CSV。
- 新增 Rust/C/WASI 嵌入示例和“诗笺台”“账册分析”两个带包依赖、测试与 API 文档的参考项目。

### 修复

- 修复标准 TOML 引号中文表名和内联依赖来源键被兼容层重复加引号的问题。
- 补齐独立 VM 对文字、完整数学、时间、文件、JSON、网络、测试以及全部新增标准模块的原生执行支持。
- 统一树解释器与 VM 的 HTTP 请求实现，并扩充标准库跨运行时一致性规格。
- 修复 VM 对泛型元组的递归运行时类型验证，并让静态模块摘要与运行时加载共用权限边界。

## 0.7.0

- `yanxu 试`新增过滤、有界并发、预期失败、逐卷超时和 JSON 机器报告。
- LSP 新增共享语义索引、补全、定义、引用、重命名、类型悬停、文档符号与 UTF-16 坐标。
- 新增 `yanxu 调试服务` DAP 适配器，支持断点、继续、单步、调用栈、作用域变量和悬停求值。
- `yanxu 文`新增声明注释、类型链接、稳定锚点、模块 API 索引和目录汇总。
- REPL 新增终端级方向键历史、Tab 会话补全、Ctrl-C 与多行行编辑。
- 新增 `yanxu 基准`，先验证双执行器输出一致，再分别报告前端和运行耗时。
- `yanxu 格`继续保证规范源码可重复解析且格式化幂等。
- 加入可单独发布的 `tree-sitter-yanxu` 语法包。
- 命令行按执行、检查、包、VM、格式化、测试、调试、文档和 LSP 分解为稳定子命令。

## 0.6.0

- 字节码不保留 AST，函数、闭包、类、协议、异常、模块和迭代降为独立原型与指令。
- VM 新增调用帧、递归闭包、方法绑定、继承、字段约束、模块执行与核心内建法。
- 新增可观测标记清扫 GC、属性内联缓存和带源文件元数据失效的模块字节码缓存。
- 加入树解释器/VM 跨实现一致性规格与性能对比入口。

## 0.5.0

- 模块顶层声明默认私有，使用 `公` 显式导出变量、常量、法与类。
- 公开变量为活绑定，模块内修改会从外部立即可见。
- 加入完整 TOML/SemVer 清单、`言序.lock`、路径/Git/中央索引依赖、校验和与离线缓存。
- 建立 `标准:<名>` 命名空间，提供文字、数学、时间、文件、JSON、网络与测试基线模块。

## 0.4.0

- 加入独立静态检查器与 `yanxu 查`，执行前检查注解、参数、归值、可变性和确定的运算符冲突。
- 类型注解支持联合、可空、函数类型和泛型容器，并进行调用推断与控制流收窄。
- 新增`协/纳`结构协议、声明式字段、公开/私有/只读/静态成员和跨模块 API 类型摘要。
