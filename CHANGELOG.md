# 变更记录

## 0.3.0-alpha.1

- 加入列、典、下标读取和下标改写。
- 加入列、文字、典的统一迭代和`逐…于…则`循环。
- 加入`定`不可改写绑定；其所指可变容器仍可修改。
- 加入`追加`、`弹出`、`有键`容器原语，并扩展`长度`。
- 加入`试／救／抛`、结构化错误值、函数与模块调用踪迹。
- 加入独立语义解析器，执行前检查重复声明、重复参数/方法、非法`归`和非法`此`。
- 版本采用 alpha 标记；源码跨度、元组和切片将在后续 0.3 alpha 完成。
- 加入包含 Docs 与下载中心的响应式官网，并支持 GitHub Pages 自动发布。
- 加入官方 VS Code 语法高亮、代码片段、运行与 REPL 扩展。
- 加入 Windows PowerShell、macOS 与 Linux 自动安装器。
- 加入三平台 CI、跨架构 Release、校验和与开源社区规范。

## 0.2.0

- 加入变量、参数与返回值的可选类型标注和运行时约束。
- 加入类、实例、动态字段、方法、`此`和`初始化`构造方法。
- 加入文件模块、命名空间、相对路径解析、缓存与循环依赖检测。
- REPL 支持多行块、持久上下文、表达式回显与交互命令。
- 加入文件执行库 API、架构说明和长期版本路线图。

## 0.1.0

- 完成词法、解析、AST 与树遍历执行链路。
- 支持基本值、变量、流程控制、一等函数、递归与闭包。
- 加入 CLI、单行 REPL、中文错误和基础示例。
