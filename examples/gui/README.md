# GUI 示例

此目录从言序多仓工作区直接引用同级`yanxu-gui`，用于核心与官方包联调：

```sh
yanxu 包 更新 yanxu-language-new/examples/gui
yanxu 查 yanxu-language-new/examples/gui/主.yx
yanxu 字节 yanxu-language-new/examples/gui/主.yx
yanxu 编 yanxu-language-new/examples/gui --release --bundle
```

单独克隆核心仓库时，使用`yanbao init 我的窗口 --gui`创建注册表依赖的项目。更多多窗口、
定时器、Canvas、图片、文件对话框、中文输入和暗色主题示例位于官方`yanxu-gui/examples`。
